About This Project

This project was created as part of the IT 3203 Web Development course at Kennesaw State University. The goal of this project is to provide an overview of the history and evolution of web browsers, from their inception to modern-day applications.

The project includes information on key concepts related to web browsers, as well as a list of resources for further reading. It is designed to be informative and accessible to a wide audience, including students, educators, and anyone interested in the history of the web.

About the Author

My name is Gavin Robinson, and I am a student at Kennesaw State University pursuing a degree in Information Technology. I have a strong interest in web development, and this project allowed me to explore the history of web browsers and design my own website.

In addition to my academic pursuits, I enjoy gaming, working on my car, and homelabbing. I am passionate about learning new technologies and applying them to real-world projects.

This website is a class project for IT 3203 Web Development.
